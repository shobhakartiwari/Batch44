# 📱 Batch 44 - iOS UIKit Learning Project

Welcome to the **Batch 44 iOS UIKit Learning Project** repository.

This project is designed to help learners build a strong foundation in iOS development using **UIKit** and **Swift**. Throughout this journey, we will cover fundamental concepts, UI development, networking, architecture patterns, and best practices used in real-world applications.

---

## 🎯 Learning Objectives

By completing this project, you will learn:

- Swift programming fundamentals
- UIKit framework
- Auto Layout and Constraints
- Storyboards and Programmatic UI
- TableView & CollectionView
- Navigation Controller
- MVC Architecture
- Networking with URLSession
- JSON Parsing and Codable
- Dependency Injection Basics
- Error Handling
- Persistence using UserDefaults
- Git & GitHub Workflow
- Clean Code Principles
